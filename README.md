## ⚡ Wat is dit?

**Wat in de Ranking** is een volledig functionele web-app gebouwd door en voor onze vriendengroep. Alles wordt gerankt — van de beste films tot de slechtste excuses, van docenten tot de vraag wie het meest *chopped* is.

> 🏆 Rank it. 🗳️ Vote on it. 📝 Review it. 🎰 Bet on it. 💣 Meme about it.

---

## 🔥 Features

### 🏆 Ranking Systeem
- **Rank It** — 1v1 battle modus: kies steeds je favoriet en ontdek je definitieve ranking
- **Live Rankings** — Real-time stemmen met anti-boosting systeem
- **Custom Categorieën** — Maak je eigen categorieën met stem-limieten
- **Bracket View** — Toernooi-stijl bracket visualisatie

### 🗳️ Polls & Reviews
- **Dagelijkse Polls** — Automatische dagelijkse vragen uit een bank van 120+ vragen
- **Docent Reviews** — Beoordeel lessen met sterren, markdown, datum & lesuur
- **Quiz Modus** — Maak quizzen met goed/fout antwoorden
- **Pin & Filter** — Pin belangrijke reviews, filter op docent of poster

### 💰 WidR Coins & Bets
- **Virtuele Currency** — Verdien coins door likes op reviews (1 like = 1 coin)
- **Startbonus** — Iedereen begint met 10 coins
- **Betting Systeem** — Zet coins in op events, win dubbel terug
- **WK 2026 Bets** — Live wedstrijden via ESPN API

### 👥 Community
- **12+ Leden** met profielen, emoji's en rollen
- **131+ Words of Wisdom** — Legendarische uitspraken uit de groepschat
- **Lore & Verhalen** — Epische momenten vastgelegd voor de eeuwigheid
- **Soundboard** — Custom geluiden uploaden en afspelen
- **Sticker Packs** — Via WhatsApp kanaal
- **Merch Store** — Officiële WidR merchandise met carousel

### 💬 Social Features
- **Berichten Systeem** — DM's tussen leden
- **Volg Systeem** — Volg je favoriete reviewers
- **Reviewer Profielen** — Klik op een naam voor stats en bio
- **Browser Notificaties** — Push-notificaties voor nieuwe berichten en polls
- **Online Status** — Zie wie er online is en op welke pagina

### 🛡️ Admin Systeem
- **Discord-stijl Rollen** — Master, Lore, Leden, Ranking, Berichten, Soundboard, Merch
- **Per-user Controls** — Verify, ban, naam-lock per gebruiker
- **Site Instellingen** — Globale toggles vanuit het admin panel
- **Wachtwoord Systeem** — Sub-admin wachtwoorden per rol

## 🛠️ Tech Stack

```
Frontend      HTML / CSS / Vanilla JS (single-page app)
Backend       Firebase Firestore (real-time database)
Auth          Firebase Auth (Google + Email/Password)
Hosting       GitHub Pages
API's         ESPN (WK 2026 wedstrijden)
```

### Architectuur
- **Single HTML file** — Alles in één `index.html` (~10.000 regels)
- **Hash-based routing** — `#home`, `#rankings`, `#polls`, etc.
- **localStorage caching** — Vermindert Firestore reads met ~75%
- **Lazy loading** — Data wordt pas geladen wanneer je naar een pagina navigeert
- **Offline-first** — Cached data wordt direct getoond
